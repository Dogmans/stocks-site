# Stock & ETF Visualization Website Structure

## Overview
This document outlines the structure, navigation, and key UI components for a website to visualize stocks, ETFs, and their metrics/historic performance. The focus is on frontend views and user experience, not backend implementation.

---

## Navigation Structure
**Header/Navbar**
  - Logo/Home
  - Search bar (for stocks/ETFs)
  - Navigation links:
    - Home
    - Markets Overview
    - Stock Screener
    - Watchlist
    - Compare
    - About

---

## Pages & Views

### 1. Home
- Brief introduction
- Featured stocks/ETFs
- Quick links to main features
- Market summary widget

### 2. Markets Overview
- Market indices performance (charts/tables)
- Top gainers/losers
- Sector performance
- News highlights

### 3. Stock/ETF Detail Page
- Name, ticker, logo
- Price chart (historical performance)
- Key metrics (market cap, PE ratio, dividend yield, etc.)
- Company/ETF description
